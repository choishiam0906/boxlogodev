## 안녕하세요, 새로운 블로그입니다

기존 티스토리 블로그(boxlogodev.com)에서 자체 개발한 Next.js 블로그로 이전했습니다.

### 왜 자체 블로그인가?

1. **커스터마이징 자유도** — 원하는 디자인과 기능을 마음껏 구현
2. **성능 최적화** — Next.js SSG로 빠른 로딩 속도
3. **개발자 경험** — MDX로 코드와 콘텐츠를 자연스럽게 결합

### 기술 스택

이 블로그는 다음 기술로 만들어졌습니다:

```typescript
const techStack = {
  framework: "Next.js 16 + App Router",
  language: "TypeScript 5 (strict)",
  styling: "Tailwind CSS v4",
  content: "Velite + MDX",
  deploy: "Vercel",
};
```

### 앞으로의 계획

- SAP ABAP 관련 기술 글
- AI/LLM 활용 사례
- 커피, 여행, 책 등 일상 이야기
